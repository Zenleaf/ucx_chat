Welcome to Ucx Chat

This is a development preview version of the software. So, you should expect some bugs. As well, a few of the features have not been implemented yet.

This version has not been integrated with the WebRTC client yet. I'll start working on that soon.

I've created a bugs channel. Please use that to report issues.

User Notes

Checkout the slash commands for some features like creating channels, inviting someone to a channel, and lots of other stuff. Just type `/` to access the commands.
